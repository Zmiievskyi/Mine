# Update Progress Tracker

Review the recent work done in this session and update the project progress documentation.

## Instructions

1. **Read current progress**: Open and review `docs/PROGRESS.md`

2. **Identify completed tasks**: Based on our conversation and any code changes, identify which tasks from the checklists have been completed

3. **Update the file**:
   - Mark completed tasks: change `- [ ]` to `- [x]`
   - Add completion date in parentheses: `- [x] Task name (YYYY-MM-DD)`
   - Update phase progress percentages in the "Current Status" table
   - Add any blockers to the "Blockers & Issues" section
   - Add any decisions made to the "Decisions Log"
   - Update the "Last updated" date at the top

4. **Update notes**: Add any relevant notes under the appropriate phase section

5. **Suggest commit**: After updating, suggest a commit with message: `docs: update progress tracker`

## Output Format

After updating, provide a summary:
- Tasks marked as complete
- New progress percentage for affected phases
- Any blockers or decisions logged
